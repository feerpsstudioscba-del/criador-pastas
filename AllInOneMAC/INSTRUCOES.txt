📁 AllInOneMAC - Versão para Mac
=====================================

Olá! Esta é a versão do All-in-One preparada para funcionar no Mac.

🎯 FUNCIONALIDADES PRINCIPAIS:
- Conversor de fontes universal (.ttf, .otf, .woff, .woff2, .svg, .eot → todos os 5 formatos)
- Criador de pastas automático
- Gerador de HTML/ZIP para banners
- Exportador de GIF
- Criador de PDF
- Sistema de presets

🚀 INSTALAÇÃO RÁPIDA:
1. Abra o Terminal no Mac
2. Navegue até a pasta AllInOneMAC
3. Execute: chmod +x install-mac.sh
4. Execute: ./install-mac.sh

🎮 USO:
1. Ative o ambiente: source .venv/bin/activate
2. Execute: npm start
